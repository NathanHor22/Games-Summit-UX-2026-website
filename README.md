# Games UX Summit 2026 - Retro TV Website

Welcome to the **Games UX Summit 2026** website! This is a retro-styled, interactive website inspired by 80s gaming aesthetics, complete with a vintage TV displaying summit information in authentic 8-bit fonts.

## 🎮 Features

### Visual Elements
- **Retro TV Display**: Classic CRT TV with Games UX Summit 2026 information
- **8-bit Typography**: Authentic pixel fonts (Press Start 2P) for that classic gaming feel
- **Vintage Effects**: TV scanlines, screen glow, and electronic noise simulation
- **Gaming Console**: Retro NES-style controller and console
- **Digital Clock**: Real-time clock display in classic green terminal font
- **Animated Wall Poster**: Classic "GAME OVER" Pac-Man themed poster

### Interactive Features
- **Power Controls**: Click the TV or console power button to toggle display
- **D-Pad Navigation**: Click the controller D-pad to cycle through different summit messages
- **Controller Buttons**: Responsive gaming controller with click effects
- **Screen Effects**: Mouse movement triggers authentic CRT flicker effects
- **Easter Egg**: Try the famous Konami code for a surprise! (↑↑↓↓←→←→BA)
